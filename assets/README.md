# Assets

Brand marks and decorative motifs for the Daffa Porto system.

| File | Purpose |
| --- | --- |
| `logo-wordmark.svg` | Primary wordmark — "Daffa Porto" in Instrument Serif italic with a signal-orange period |
| `logo-monogram.svg` | Compact `dp.` monogram for favicons & avatars (96×96) |
| `favicon.svg` | 32px inverse-on-ink favicon variant |
| `signal-dot.svg` | The 16px filled circle status pip — "shipped" / "in progress" indicator |
| `sparkline.svg` | Decorative sparkline used on project cards and slide backgrounds |

## Status

⚠️ **No user-provided logo or visual assets.** Everything here was generated by the design system at first build. Drop a real logo into this folder and update `logo-wordmark.svg` / `logo-monogram.svg` to swap.

If product screenshots become available (`santra.id`, `RUNA`, etc.), put them in `assets/screenshots/` and reference them from the portfolio kit.
